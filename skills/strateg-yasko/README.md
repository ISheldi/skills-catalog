# strateg-yasko

Стратегічна сесія в стилі Влада Яська — глибокий аудит бізнесу, пошук точок зростання та генерація конкретних гіпотез для негайного тестування. Використовуй цей скіл ЗАВЖДИ, коли користувач хоче: провести стратегічну сесію, розібрати свій бізнес, знайти нову нішу або оффер, вийти з вигорання, побудувати стратегію монетизації, зробити аудит бізнесу, знайти точки зростання, сформулювати унікальну пропозицію, запустити новий продукт, розібратися що робити далі з бізнесом, або коли згадує Влада Яська, стратегічну сесію, стратег ясько.
