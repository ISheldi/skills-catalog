Что под капотом

На случай если интересно, на чём это работает.

Whisper — это нейросеть от OpenAI для распознавания речи. whisper.cpp — её портированная версия на C++, которая работает напрямую на твоём процессоре. На Apple Silicon (M1/M2/M3/M4/M5) она использует Metal — это технология ускорения вычислений на видеокарте Mac, делает работу в 5–10 раз быстрее.

Модель large-v3-turbo — это последняя и самая быстрая версия Whisper большого размера. Качество как у large-v3, но в 8 раз быстрее. Для русского языка — топовый выбор.

Как этим пользоваться

Три шага после установки. Делается за пару минут на любой чат.

  1. Экспорт чата из Telegram Desktop

    Открой нужный чат на компе → правый верхний угол ⋯ → Export chat history. В настройках поставь галочку Voice messages и формат HTML. Жми Export.

    Папка с экспортом появится в ~/Downloads/Telegram Desktop/ChatExport_YYYY-MM-DD/

  2. Открой Claude Code и попроси расшифровать

    В терминале запусти claude и напиши:

    расшифруй голосовые из ~/Downloads/Telegram Desktop/ChatExport_2026-04-29

    Скилл подхватится сам. Клод увидит триггер «расшифруй голосовые» и запустит транскрибацию.

  3. Получи готовый текст

    В папке экспорта появится подпапка transcripts/:

    • audio_*.txt — отдельные файлы для каждого голосового
    • _ALL.md — общий файл со всем подряд

    Дальше можешь скормить _ALL.md Клоду и попросить выжимку, инсайты, задачи — то, для чего ты вообще эти голосовые расшифровывал.

Что может пойти не так

FAQ и решения проблем.

У меня Intel Mac, сработает?
Сработает, но медленнее в 5–10 раз — Metal-ускорение работает только на Apple Silicon (M1/M2/M3/M4/M5). На Intel оно пойдёт через CPU. Если у тебя M-чип — летает.
У меня Windows / Linux
Этот скилл — только для Mac. На Windows/Linux есть аналогичные пути через WSL или нативные сборки whisper.cpp, но это уже отдельная история.
Что такое Claude Code и где его взять?
Claude Code — это инструмент от Anthropic для работы с Клодом, через него мы делаем 90% задач на курсе. Если ты на 2-м потоке AI-Маркетолога — он у тебя уже установлен и настроен (мы ставили на первом занятии). Если нет — этот скилл не для тебя, он работает только в связке с Claude Code.
Куда вообще писать команду установки?
В то же окно Claude Code, где ты обычно общаешься с Клодом. Открываешь Claude Code как на занятиях, скопированный текст вставляешь как обычное сообщение, жмёшь Enter. Дальше Клод сам всё сделает, ты только подтверждаешь его запросы кнопкой Yes.
Сколько места займёт?
Модель — 1.5 ГБ. Зависимости (whisper-cpp + ffmpeg) — около 200 МБ. Скрипт и скилл — пара килобайт. Итого ~1.7 ГБ.
Установка сломалась, что делать?
Сделай скриншот того, что Клод вывел красным — и пришли в чат AI-Маркетолога. Чаще всего проблема — нет интернета на этапе скачивания модели (1.5 ГБ) или Homebrew просит пароль (Клод покажет это сообщение). В обоих случаях скажи Клоду «запусти установку ещё раз» — скрипт подхватит с того места, где упал, ничего повторно не качается.
А кружочки (видео-сообщения)?
Подхватываются автоматически, если в экспорте поставить галочку Video messages. Скилл идёт по папке round_video_messages/ и достаёт оттуда аудиодорожку.
Можно через терминал без Claude?
Да. Инсталлятор кладёт команду в ~/bin/transcribe-tg.sh. Запуск:
~/bin/transcribe-tg.sh "путь/к/ChatExport_..."
Как удалить, если не понравится?
Выполни в терминале:
brew uninstall whisper-cpp ffmpeg
rm -rf ~/.whisper-models ~/.claude/skills/telegram-transcribe ~/bin/transcribe-tg.sh
