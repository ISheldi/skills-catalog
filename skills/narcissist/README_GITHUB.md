## 📖 Описание

**Narcissist Communication Skill** — это интеллектуальный агент для Claude Code, который помогает адаптировать сообщения для безопасного и эффективного общения с людьми с нарциссическим расстройством личности.

### 🎯 Что делает skill:

- ✅ **Обнаруживает опасные фразы** — автоматически находит триггеры нарциссической ярости
- ✅ **Адаптирует сообщения** — переформулирует на язык фактов, убирает эмоции
- ✅ **Подбирает паттерны** — использует проверенные речевые стратегии
- ✅ **Интегрируется с NotebookLM** — доступ к экспертной базе знаний
- ✅ **Даёт объяснения** — почему именно так работает

## 🚀 Быстрый старт

### Установка

1. **Скачай и распакуй:**
   ```bash
   # Вариант 1: Скачай ZIP
   wget https://github.com/vladyasko/narcissist-communication-skill/archive/refs/heads/main.zip
   unzip main.zip

   # Вариант 2: Клонируй репозиторий
   git clone https://github.com/vladyasko/narcissist-communication-skill.git
   ```

2. **Установи в Claude Code:**
   ```bash
   cp -r narcissist-communication-skill ~/.claude/skills/narcissist-communication
   ```

3. **Готово!** Перезапусти Claude Code

### Использование

**Способ 1: Интерактивный режим**
```
/narcissist
```
Claude запросит цель, контекст и черновик сообщения.

**Способ 2: Быстрый режим**
```
/narcissist "Нет, я не могу дать тебе деньги, потому что ты не вернул прошлый раз"
```

**Способ 3: JSON формат**
```
/narcissist {"goal": "отказать в займе", "draft": "не дам"}
```

## 🎯 Возможности

### Автоматическое обнаружение опасных элементов

Skill проверяет ваши сообщения на:

- **13 запрещённых фраз** (триггеры ярости)
- **Паттерны оправданий** ("потому что", "из-за того что")
- **Эмоциональные слова** (боль, обида, устал)
- **Прямые обвинения** ("ты всегда", "ты никогда")

### 4 типа ситуаций

Skill адаптирует сообщения для:

1. **Просьба** — когда нужно что-то попросить
2. **Отказ** — когда нужно сказать "нет"
3. **Переговоры** — когда нужно договориться
4. **Конфликт** — когда нарцисс провоцирует

### Интеграция с NotebookLM (опционально)

Для глубокого анализа skill может обращаться к экспертной базе знаний:
- Материалы психолога Виталия Филиппова
- Проверенные техники коммуникации
- Стратегии защиты от манипуляций

## 🛡️ Золотые правила коммуникации

### ❌ НИКОГДА НЕ ГОВОРИ:

| Фраза | Почему опасно |
|-------|---------------|
| "Я тебя понимаю" | Воспринимается как "Я вижу тебя насквозь" |
| "Решай сам" | "Ты пустое место" → ярость |
| "Извинись" | Активирует стыд → агрессия |
| "Ты сделал мне больно" | Даёт карту уязвимостей |

### ✅ ВСЕГДА ДЕЛАЙ:

- Говори на языке **фактов**, не эмоций
- Сохраняй **спокойствие**
- Используй **паузы** ("мне нужно подумать")
- Апеллируй к **грандиозности**
- Давай **иллюзию контроля**
