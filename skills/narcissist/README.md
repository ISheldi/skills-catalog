# Narcissist Communication Skill

Агент для эффективной коммуникации с людьми с нарциссическим расстройством личности.

## Описание

Этот skill помогает адаптировать твои сообщения для общения с нарциссом, используя:
- Базу знаний из NotebookLM (психолог Виталий Филиппов)
- Проверенные паттерны коммуникации
- Автоматическое обнаружение опасных фраз-триггеров

## Использование

### Способ 1: Команда без аргументов (интерактивный режим)
```
/narcissist
```

Skill запросит:
1. Цель общения
2. Контекст (опционально)
3. Черновик сообщения

### Способ 2: Быстрый режим (передать черновик сразу)
```
/narcissist "Нет, я не могу тебе дать деньги, потому что ты не вернул прошлый раз"
```

### Способ 3: JSON формат (полная информация)
```
/narcissist {"goal": "отказать в просьбе о займе", "context": "уже давал деньги 3 раза, не возвращал", "draft": "Нет, не дам"}
```

## Что делает skill

1. **Анализирует** твой черновик на опасные элементы:
   - Запрещённые фразы (триггеры ярости)
   - Оправдания и объяснения
   - Эмоциональные слова

2. **Определяет тип ситуации:**
   - Просьба (нужно что-то попросить)
   - Отказ (сказать "нет")
   - Переговоры (договориться)
   - Конфликт (нарцисс провоцирует)

3. **Адаптирует сообщение** используя:
   - Правильные речевые паттерны
   - Язык фактов вместо эмоций
   - Техники защиты от манипуляций

4. **Выдаёт результат:**
   - Готовое адаптированное сообщение
   - Объяснение почему это работает
   - Альтернативные варианты
   - Паттерны для данной ситуации

## Примеры

### Пример 1: Отказ в просьбе

**Твой черновик:**
```
Нет, я не могу дать тебе деньги, потому что ты мне ещё с прошлого раза не вернул
```

**Адаптированное сообщение:**
```
Нет, я не буду этого делать сейчас. Я занят(а) своими делами, а к этому вернусь позже.
```

**Почему опасно:**
- "потому что" → оправдание = слабость
- Упоминание прошлого → провокация конфликта

---

### Пример 2: Просьба

**Твой черновик:**
```
Извинись передо мной за вчерашнее! Ты меня обидел!
```

**Адаптированное сообщение:**
```
Ты всегда такой сильный и уверенный. Думаю, такой классный человек, как ты, может признать, что вчерашняя ситуация прошла не совсем гладко.
```

**Почему опасно:**
- "Извинись" → активирует стыд → ярость
- "Ты меня обидел" → даёшь карту уязвимостей

---

## Интеграция с NotebookLM

Skill автоматически использует базу знаний из NotebookLM:
- Для стандартных ситуаций использует готовые паттерны
- Для сложных случаев запрашивает дополнительную информацию

База знаний должна быть активирована:
```bash
cd ~/.claude/skills/notebooklm
python3 scripts/run.py notebook_manager.py activate --id "психология-нарциссизма---виталий-филиппов"
```

## Запрещённые фразы (автоматически обнаруживаются)

- "Я тебя понимаю"
- "Решай сам" / "Мне всё равно"
- "Извинись" / "Признай вину"
- "Давай честно" / "Скажи правду"
- "Ты сделал мне больно"
- "Ты нарцисс"
- "Почему ты..."
- "Ты всегда..." / "Ты никогда..."

## Золотые правила

✅ **ДЕЛАЙ:**
- Говори на языке фактов
- Сохраняй спокойствие
- Используй паузы ("мне нужно подумать")
- Апеллируй к его грандиозности
- Давай иллюзию контроля

❌ **НЕ ДЕЛАЙ:**
- Не оправдывайся
- Не объясняй ему его поведение
- Не показывай эмоции (это его топливо)
- Не пытайся его переубедить

## Troubleshooting

**Проблема:** Skill не находит NotebookLM
**Решение:** Проверь путь в `narcissist_agent.py` (переменная `NOTEBOOKLM_PATH`)

**Проблема:** Нет доступа к базе знаний
**Решение:** Активируй ноутбук:
```bash
cd ~/.claude/skills/notebooklm
python3 scripts/run.py notebook_manager.py activate --id "психология-нарциссизма---виталий-филиппов"
```
