# 🚀 Quick Start Guide - Narcissist Communication Skill

## Установка завершена! ✅

Skill уже настроен и готов к работе.

---

## 📖 Как использовать (3 способа)

### 1️⃣ Интерактивный режим (РЕКОМЕНДУЕТСЯ)

Просто напиши в Claude Code:
```
/narcissist
```

Claude запросит:
- Твою цель
- Контекст (опционально)
- Черновик сообщения

Затем выдаст адаптированное сообщение с объяснением.

---

### 2️⃣ Быстрый режим

Передай сообщение сразу:
```
/narcissist "Твой черновик сообщения"
```

**Пример:**
```
/narcissist "Нет, я не могу дать тебе деньги, потому что ты не вернул прошлый раз"
```

Получишь:
- Анализ опасных фраз
- Адаптированное сообщение
- Объяснение почему это работает

---

### 3️⃣ Продвинутый режим (JSON)

Для максимального контроля:
```
/narcissist {"goal": "отказать в займе", "context": "уже давал 3 раза", "draft": "не дам"}
```

## 💡 Примеры использования

### Пример 1: Отказ в просьбе

**Ты пишешь:**
```
/narcissist "Извини, не могу помочь, потому что у меня самого проблемы"
```

**Получаешь:**
```
⚠️ ОПАСНО:
- "извини" → не нужны извинения за свои границы
- "потому что" → оправдание = слабость
- "проблемы" → даёт материал для манипуляций

✅ АДАПТИРОВАННОЕ:
"Нет, я не буду этого делать сейчас."

🧠 ПОЧЕМУ РАБОТАЕТ:
- Твёрдое "нет" без оправданий
- Нет эмоций и уязвимостей
- Факт вместо объяснений
```

---

### Пример 2: Попросить что-то

**Ты пишешь:**
```
/narcissist "Извинись передо мной за вчерашнее!"
```

**Получаешь:**
```
⚠️ ОПАСНО:
- "Извинись" → активирует стыд → ярость

✅ АДАПТИРОВАННОЕ:
"Ты всегда такой сильный и уверенный. Думаю, такой классный человек, как ты, может признать, что вчерашняя ситуация прошла не совсем гладко."

🧠 ПОЧЕМУ РАБОТАЕТ:
- Апелляция к грандиозности
- Нет прямого требования
- Он делает это "сам"
```

---

### Пример 3: Конфликт

**Ты пишешь:**
```
/narcissist "Почему ты всегда на меня кричишь?!"
```

**Получаешь:**
```
⚠️ ОПАСНО:
- "Почему" → втягивание в бессмысленные объяснения
- "ты всегда" → обобщение = провокация
- Эмоция → его топливо

✅ АДАПТИРОВАННОЕ:
"Я вижу, что происходит. Интересно, что конкретно вызвало у тебя такую реакцию?" [спокойно]

🧠 ПОЧЕМУ РАБОТАЕТ:
- Называние манипуляции
- Спокойствие разрывает шаблон
- Аналитический вопрос вместо эмоций
```

- ✅ Предстоит сложный разговор
- ✅ Хочешь отказать безопасно
- ✅ Нужно попросить о чём-то
- ✅ Нарцисс провоцирует конфликт
- ✅ Хочешь проверить свой черновик

---

## 🔥 Pro Tips

1. **Не торопись отправлять** — создай паузу между получением сообщения и ответом
2. **Используй публичность** — важные разговоры веди при свидетелях
3. **Сохраняй спокойствие** — эмоции = топливо для нарцисса
4. **Фиксируй факты** — записывай договорённости
5. **Не объясняй "почему"** — чем меньше информации, тем лучше

## 🆘 Если что-то не работает

### Ошибка: "ModuleNotFoundError"
**Решение:** Skill использует Python 3, проверь установку:
```bash
python3 --version
```

### Ошибка: "NotebookLM not found"
**Решение:** Убедись что NotebookLM skill установлен:
```bash
ls ~/.claude/skills/notebooklm
```

### Skill не появляется в списке
**Решение:** Перезапусти Claude Code
