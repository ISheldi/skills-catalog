# seedance-fashion-lookbook

Generate fashion lookbook, model showcase, and style video prompts for Seedance 2.0 on Higgsfield. Use whenever the user wants fashion video content, lookbook videos, model walks, outfit showcases, style guides, fashion campaigns, runway clips, streetwear content, or any fashion/clothing video. Triggers on: fashion video, lookbook, model showcase, outfit, style guide, fashion campaign, runway, streetwear, collection launch, fashion ad, clothing video, OOTD, fashion film, or any fashion/model video request. Use even for "show off this outfit" or "fashion content for my brand.

## Установка
```bash
git clone https://github.com/yasikvlad/seedance-fashion-lookbook ~/.claude/skills/seedance-fashion-lookbook
```
