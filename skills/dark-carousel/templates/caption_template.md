# Template — Подпись под пост

## Структура

```
[1 строка — крючок / провокация / вопрос]

[3-5 строк — суть, без дублирования слайдов]

[1-2 строки — авторская позиция / личный угол]

[Мягкий призыв или вопрос аудитории]
```

## Правила

- **Первое слово НЕ «Я».** Начни с глагола, вопроса, факта.
- **Без хэштегов.**
- **Без рублей** — только `$` или без валюты (см. `knowledge/known_bugs.md` баг 4).
- Не дублируй текст со слайдов — добавь контекст или мета-комментарий.
- Авторская позиция — личный угол, не «вода».
- Призыв мягкий: «А у тебя как?», «Что выбираете?», «Какая часть ближе?».

## Пример наполнения

```
Когда модель «работа = успех» сломалась, я заметил странное.

Половина моих знакомых ушли в кочевую жизнь и нашли там покой,
другая — в стартапы и нашли там смысл.
Но обе группы перестали считать выходные «отдыхом».
Отдых стал тем, чем должен быть — состоянием, а не временем.

Для меня граница теперь не «работаю / не работаю»,
а «включён / отпустил».

А у вас где сейчас эта граница?
```
