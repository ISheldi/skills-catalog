# Visual DNA — Формат A (Бренд-личность)

**Когда использовать:** темы про личный бренд, инсайты, философию работы, путь автора, экспертность.

## Палитра

- Фон: `#EFEAE0` (cream-beige), без клетки
- Акцент: красный `#E63A1F`
- Текст: чёрный `#1A1A1A`, белый `#FFFFFF`

## Типографика

- Основной шрифт: **Inter**
- Заголовки: weight `900`
- Текст: weight `700-800`
- Системные macOS + Google Fonts (Playwright resolve)

## Структура

- **11-12 слайдов**
- Фото автора на слайдах **1, 5, 9** (каждые 3-4 слайда)
- Слайд 1 = фото лица + интрига (обложка)
- Слайд 11 — финальный CTA, ник по центру крупно

## Запретные зоны

- Правый верх — счётчик IG ❌
- Правый низ — звук IG ❌

## Обязательные элементы

- Левый верх: `NN / 11` (Lato Bold 24px, серый или светлый на фото)
- Левый низ: `@vladyasko` (Lato Bold 26px) — на каждом слайде кроме финала

## Шаблоны и скрипты

- HTML-шаблон: `templates/slide_format_A.html` (внутри этого скилла)
- Рендер: каждая карусель собирает свой `generate_slides.py` в `готовые/<название>/черновики/` на базе эталона `examples/Карусель Архетипы/generate_slides.py` (Playwright → PNG 1080×1350)
- Дизайн-параметры: `config/settings.yml` (источник правды) + этот файл
- Точка входа Claude Code: `Карусели/CLAUDE.md`

## Технология

- Фото-слайды → Higgsfield MCP (`nano_banana_2`) с фото-референсом
- Типографские слайды → HTML + Playwright (системный Inter + Google Fonts)
