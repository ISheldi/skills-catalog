# Instagram UI — Запретные зоны

При дизайне слайда учитывай, что Instagram оверлеит свой интерфейс поверх изображения. В эти углы НЕ ставим текст или важные элементы.

| Угол | Что там у IG | Куда ставим текст |
|---|---|---|
| **Левый верх** | свободно | bookmark + счётчик `NN/TT` |
| **Правый верх** | счётчик слайдов IG ❌ | **пусто** |
| **Правый низ** | иконка звука IG ❌ | **пусто** |
| **Левый низ** | свободно | ник `@vladyasko` ✅ ВСЕГДА |

## Правила

- **Ник `@vladyasko`** — на каждом слайде в левом нижнем углу.
- **Исключение:** финальный CTA-слайд — ник может быть по центру крупно (тогда в углу не дублируется).
- **Нумерация `NN/TT`** — верхний левый. Шрифт `JetBrains Mono` для стабильной ширины символов (фикс бага 1).
