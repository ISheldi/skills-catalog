# Visual DNA — Формат B (Гибрид Higgsfield + HTML)

**Когда использовать:** темы со скринами / схемами / списками брендов / "топ-N" / архетипы / репозитории / сравнения. Когда лицо автора НЕ нужно на каждом слайде.

## Палитра

- Фон: `#EFEAE0` (cream)
- Сетка/grid: `#CFC4A8`, шаг 40px (анимированный медленный drift)
- Bookmark/закладка: оранжевая `#E85A2C` (top-left)
- Карточки: чёрные `#0A0A0A`, border-radius 24px, оранжевая тень
- Акцент: `#E85A2C` (тёплее, чем формат A)

## Типографика

- Заголовки: **Inter** weight 900
- Body: **Inter** weight 600-700
- Счётчики/моноширинные элементы: **JetBrains Mono**

## Структура

- 1 (cover, Higgsfield generation) + N контентных + 1 dark-side / CTA
- Слайд 1 = отдельная Higgsfield-генерация (статуя / визуал / 3D-объект)
- Контентные слайды = HTML/CSS/Playwright (можно с анимациями → MP4)

## Эталон формата B (2026-05-05)
