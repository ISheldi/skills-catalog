# ИСВП — формулы хуков для обложек

> ИСВП = **И**нтрига + **С**рочность + **В**ыгода + **П**ричастность.
> Это четыре «рычага» внимания. Хук обложки — обычно комбинация 2–3 из них.
> Источник: `материалы/Carousel_Maker.pdf` + `материалы/ИСВП_универсальная_таблица.pdf` + `материалы/ИСВП_готовые_хуки.pdf` (для конкретных шаблонов под нишу).

---

## ТОП-5 КОМБИНАЦИЙ

### И + С — Интрига + Срочность *(самая сильная для охвата)*
Шаблон:
```
[Провокационное утверждение о теме]
Пока ты не знаешь [X] — ты теряешь [Y]
```
Пример:
> ДЕНЬГИ ОБЕСЦЕНИВАЕТ НЕ ИНФЛЯЦИЯ
> подпись: пока ты винишь экономику — твои деньги уходят к тому, кто понимает разницу

### В + П — Выгода + Причастность *(для конкретного результата)*
Шаблон:
```
Как [конкретная ЦА] получает [результат]
[Метод] без [боль/барьер]
```
Пример:
> КАК ФРИЛАНСЕР БЕРЁТ $5K
> ЗА ПРОЕКТ
> подпись: без агентства, без команды, без выгорания

### И + В — Интрига + Выгода *(для образовательного контента)*
Шаблон:
```
[Контринтуитивный факт или вопрос]
И вот что это меняет на практике
```
Пример:
> МЕНЬШЕ КОНТЕНТА = БОЛЬШЕ ОХВАТ
> подпись: парадокс, который доказали 60k подписчиков моего канала

### С + П — Срочность + Причастность *(для трендовых тем)*
Шаблон:
```
Если ты [признак ЦА] — это происходит прямо сейчас
[Факт / угроза / возможность]
```
Пример:
> ЕСЛИ ТЫ SMM-ЩИК
> ЭТО УЖЕ ПРОИСХОДИТ
> подпись: 80% постов в Instagram пишет AI. где будешь ты через год?

### И + С + В — тройной удар *(для максимального вовлечения)*
Шаблон:
```
[Неожиданная правда]
[Почему это срочно]
[Что ты получишь понимая это]
```

## ЭЛЕМЕНТЫ ПО ОТДЕЛЬНОСТИ — ДЛЯ КОМБИНИРОВАНИЯ

### И — Интрига
- **Contradiction:** «Все говорят X — реальность Y»
- **Knowledge gap:** «Есть один нюанс в [теме] который меняет всё»
- **Provocative question:** «Что если [распространённое убеждение] — ложь?»

### С — Срочность
- «Пока ты читаешь это — [мир/конкуренты] уже [делают X]»
- «Каждый день без [X] ты теряешь [Y]»
- «Те кто уже [сделал X] — [получают Y]»

### В — Выгода
- «Как [глагол] за [время/ресурс]»
- «[Действие] = [конкретный результат]»
- «Из [состояния A] в [состояние B]»

### П — Причастность
- «Если ты [признак ЦА] — это для тебя»
- «Для тех кто [верит / строит / работает в X]»
- «Ты тоже [сталкивался с X]?»

## ШАГ 4А ИЗ ИНСТРУКЦИИ — 5 ВАРИАНТОВ ХУКА

При генерации обложки **всегда давать пользователю 5 вариантов** с разными углами:

| Угол | На каком ИСВП-рычаге | Когда работает |
|---|---|---|
| 1. Провокация против общепринятого | И (Contradiction) + С | вирусность, охват |
| 2. Личная история / признание | П + И (story) | вовлечение, эмоция |
| 3. Вопрос-зеркало читателю | П + И (question) | сохранения, узнавание |
| 4. Контрарианский совет | И + В | контент-эксперт, авторитет |
| 5. Сравнение / парадокс | И + В + С | максимальный сторителлинг |

Для каждого варианта в ответе пользователю: `Hero обложки`, `Caption обложки`, 1–2 строки куда пойдёт вся карусель.
