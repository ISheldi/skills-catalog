# Материалы для каруселей

Этот каталог содержит справочные материалы. Читай их по необходимости (не все сразу — экономь токены).

## MD-файлы (быстрые выжимки для агента)

| Файл | Когда читать |
|---|---|
| `ца.md` | Перед каждой каруселью под инфобиз — чтобы попасть в боли и триггеры аудитории. Для личного бренда — справочно. |
| `исвп-формула.md` | На шаге «Сформулировать 5 вариантов хука» — оттуда берутся комбинации. |
| `title-frameworks.md` | Если карусель «полезная» (как-сделать / рейтинг / сравнение) — оттуда формула для заголовка. |

## PDF — оригиналы (читать только когда нужны конкретные примеры)

| Файл | Что внутри |
|---|---|
| `Carousel_Maker.pdf` | Базовая структура слайдов и текстов |
| `ИСВП_универсальная_таблица.pdf` | Полная матрица ИСВП-комбинаций с примерами |
| `ИСВП_готовые_хуки.pdf` | Готовые хуки под разные ниши — копируй и адаптируй |
| `Title_Frameworks_Strategy.pdf` | Источник 10 английских формул из `title-frameworks.md` |

## Шрифты

`шрифты/Barlow_Condensed/` — Latin-only набор Barlow Condensed (нет кириллицы). Использовать только для английских каруселей.

Для русских hero — системные шрифты macOS (Helvetica Neue Black, SF Pro Display Black) либо веб-шрифты через CSS @import: Anton, Druk Wide Heavy, Bebas Neue.

## ЗАЧЕМ MD-ВЫЖИМКИ ЕСЛИ ЕСТЬ PDF

PDF — для людей. MD — для агентов:
- быстрее парсятся
- лучше индексируются Claude
- проще диффить и редактировать
- их можно процитировать в чате компактно

Поэтому при работе сначала читай MD. PDF — резерв для глубоких ссылок (например: «Я предлагаю формулу X — она из таблицы И+С на стр. 4 PDF»).
