# Библиотека видео-фонов — DARK Carousel

Клипы для «живых» слайдов (видео-фон вместо статичной сетки). Все клипы должны быть приведены к **1080×1350, ≤15 сек, H.264 + AAC + faststart** (IG-совместимы).

> 📦 **Эта библиотека пустая** — оригинальные клипы были личным видео автора скилла и в дистрибутив не входят. Наполни её своими видео.

## Как наполнить свою библиотеку

1. Закинь исходники (любой формат/соотношение) в `raw/`.
2. Запусти обработку: `"$VENV/bin/python" scripts/process_video_bg.py` — приведёт к 1080×1350 ≤15с H.264 и сделает превью-thumbnails.
   - `$VENV` — путь к venv скилла (см. `ONBOARDING.md`, плейсхолдер `/Users/vladyasko/.claude/skills/dark-carousel/.venv`).
3. Скрипт создаст `video_bg_index.json` (машинный индекс) и `.thumbs/*.jpg` (превью).
4. Протегируй клипы в `video_bg_index.json`: `mood` (calm / bold / dynamic / warm / focused), `scene` (nature / city / talking_head / b_roll), `recommended_for` (intro / thesis / context …).

## Как использовать на слайдах

См. `workflows/video_backgrounds.md`. Кратко: 2–4 слайда на карусель (НЕ обложка, обычно НЕ финальный CTA), подбор клипа рандомом по тегам без повторов, на видео — только светлый текст + затемнение (scrim).

## Структура папки

```
video_bg/
├── INDEX.md              ← этот файл
├── raw/                  ← сюда кладёшь исходники
├── *.mp4                 ← обработанные клипы (появятся после запуска скрипта)
├── .thumbs/*.jpg         ← превью (появятся после запуска скрипта)
└── video_bg_index.json   ← теги (появится после запуска скрипта)
```
