# dark-carousel — Instagram Content Factory

Скилл для производства Instagram-каруселей в DARK editorial-стиле.

> Создано **by Stanislav Tsys** · Instagram [@stastsis](https://instagram.com/stastsis)

🆕 **Ставишь впервые?** Смотри `ONBOARDING.md` — установка и настройка за 5 минут.

## Что это

Тонкий оркестратор (`SKILL.md`) + модульная база знаний:
- `knowledge/` — статичные знания (визуальные DNA форматов, баги, формулы хуков, спеки анимаций)
- `prompts/` — преамбулы для Higgsfield (фото / типографика / 3D-объект)
- `workflows/` — пошаговые сценарии (онбординг, план+апрув, рендер, итерация Higgsfield)
- `templates/` — HTML-шаблоны слайдов + caption-template
- `config/settings.yml` — параметры по умолчанию (тут же твой ник)
- `scripts/` — Python-скрипты (рендер, генерация)
- `materials/` — шрифты, PDF-методички, формулы хуков, библиотека видео-фонов
- `examples/` — готовый референс-карусель «Архетипы» (формат B + анимации)

## Для кого

- Любой, кто ведёт Instagram и хочет быстро собирать качественные карусели в едином стиле.
- Темы под которые заточен стиль: AI, продуктивность, личный рост, тревел, философия (но подойдёт под любой контент).

## Как запускать

В Claude Code:
```
/carousel
```
или просто пиши: «карусель», «слайды», «новый пост», «давай карусель».

Skill сам спросит источник (текст / URL / идея), формат (A — бренд-личность / B — гибрид Higgsfield+HTML), объём (8/12/16/20+) и стиль хука. Потом выдаст полный план на апрув и пойдёт рендерить.

## Настройка перед использованием

Скилл переносимый — замени плейсхолдеры под свою систему (детали в `ONBOARDING.md`):
