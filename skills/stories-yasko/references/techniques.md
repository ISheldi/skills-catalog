# Техники создания сторис — метод Яско

## 1. 20 принципов подачи

1. **Интригующее начало** — вопрос или Life-кадр. Тема ясна за ≤ 3 сторис.
2. **Конкретная проблема + contrast** — «Как заработать 200K? Но без команды и бюджета.»
3. **Структура** — покажи, что получит зритель: «3 ошибки + формула + лайфхак»
4. **Proof рано** — скриншот кейса ДО объяснения методологии
5. **Конкретика** — «Однушка в Москве, 15 млн» вместо «квартира»
6. **Формулы** — «200K = 40 продаж × 5000₽»
7. **Диалогичность** — «И ты думаешь: 'Ладно, а где взять охваты?'»
8. **Варианты** — «Если ты эксперт → свой курс. Если нет → найди эксперта 50/50»
9. **Проверка понимания** — «Тут понятно?» + кнопки Да/Нет
10. **Дробление целей** — большая цель → маленькие шаги
11. **Честность** — «Не скажу, что просто. Но реально.»
12. **Баланс** — сложность + достижимость
13. **Подкрепление опытом** — своя история, скрин из кабинета
14. **Уверенные гарантии** — «Готов поспорить — формула работает.»
15. **Секретный ингредиент** — уникальное название метода (term brand)
16. **Эксклюзивность** — «Не для всех. Узкий круг.»
17. **Цена в конце** — после раскрытия ценности
18. **Лёгкое завершение** — юмор, самоирония
19. **Итоговое резюме** — «ИТОГО: 1... 2... 3...»
20. **Walk-away value** — зритель может УЙТИ и СДЕЛАТЬ что-то

## 2. Визуальное оформление

### Цвета фона
| Цвет | Для чего |
|---|---|
| Чёрный | Заголовки КАПСОМ, важные концепции |
| Красный | Проблемы, ошибки |
| Зелёный | Решения, правильные подходы |
| Белый/светлый | Основной текст, пояснения |

### Текст
- Заголовки: КАПС, крупный шрифт
- Списки: эмодзи как маркеры (🎯 ❌ ✅ 🔍 ❤️)
- Основной: 1-3 коротких предложения, 5-10 слов на строку
- Акценты: жирный, другой цвет, числа крупнее

### Локации
- Контрастные, запоминающиеся
- Чередовать фоны
- Тёмный фон для формул

## 3. Вовлечённость

### Контрасты
- Таблицы «правильно vs неправильно»
- Метафоры: «ПРОДАВАТЬ КАК ДЕВСТВЕННИК»

### Интрига
- Неполная информация → зритель достраивает
- Провокационные заголовки

### Proof
- Скриншоты переписок после каждого блока
- Реакции аудитории, кейсы с цифрами

### Интерактив
- В первых 5 сторис — минимум 1
- Далее каждые 5-7 сторис
- После сложных блоков — обязательно

---

## 4. Содержательные приёмы

### Формулы
```
200K = 40 продаж × 5000₽
40 продаж = 3% × 1300 охватов
```

### Проблема → Признаки → Решение
1. Заголовок КАПСОМ
2. Признаки (списки с эмодзи)
3. Решение
4. Proof (скрин/кейс)

### Эмоциональное вовлечение
- «Боль» от проблемы
- Образ «светлого будущего»
- Личные истории
- «Хочу быть похожим» — ключ к продажам

---

## 5. Текст на сторис vs Речь голосом

| | Текст на экране | Речь голосом |
|---|---|---|
| Длина | 5-15 слов | 15-30 секунд |
| Функция | Краткий тезис, якорь | Объяснение, нюансы |
| Стиль | КАПС, заголовочный | Разговорный, живой, в ToV Яско |

**Правило:** Текст и голос ДОПОЛНЯЮТ, не дублируют. Текст = «заголовок», голос = «статья».
Голос = как человек РЕАЛЬНО говорит: с паузами, «короче», энергией.
