# Стилистический ключ каруселей Влада Яско

## Концепция

Минимализм + контраст. Чёрный фон — это не пустота, это акцент.
Каждый слайд дышит. Текст — единственный визуальный элемент на текстовых слайдах.
Обложка останавливает скролл через сильную фотографию с тёмным оверлеем.

Референс-стиль: карусели Анжелы (подруга Влада) — проверенная рабочая формула.

---

## ВИЗУАЛЬНЫЙ СТАНДАРТ

### Технические параметры
- **Формат:** 9:16 вертикаль, 1080×1920px
- **Соотношение сторон Instagram:** 4:5 (обрезка под фид — 1080×1350px)

### Цветовая палитра
```
Фон слайдов:    #000000  (чистый чёрный)
Основной текст: #FFFFFF  (белый)
Акцентный цвет: #F5C518  (тёплый жёлтый / золотой)
Оверлей фото:   rgba(0,0,0,0.60)  (60% затемнение)
```

### Типографика
```
Заголовок обложки:  Inter ExtraBold / Neue Haas Grotesk Bold
                    UPPERCASE, белый, крупный (≈72-90pt)
                    Максимум 3 строки

Акцент (жёлтый):    Inter SemiBold, sentence case
                    Используется для 1-2 ключевых фраз на слайд

Тело текста:        Inter Regular, белый, sentence case
                    18-22pt, межстрочный 1.5
```

### Обложка (Слайд 0)
- Full-bleed фото человека — уверенная поза, читаемое лицо
- Тёмный оверлей 55-65% поверх фото
- Текст позиционируется в нижней 35-40% слайда
- Структура текста обложки:
  ```
  [ГЛАВНЫЙ ЗАГОЛОВОК UPPERCASE]     ← белый, ExtraBold
  [ЖЁЛТЫЙ АКЦЕНТ — короткая фраза]  ← жёлтый #F5C518
  [Подтекст обычным шрифтом]        ← белый, Regular
  ```

### Текстовые слайды
- Фон: чистый чёрный #000000
- Текст: белый, regular weight
- 1 слайд = 1 мысль
- Максимум 4-5 строк текста
- Ключевые слова/фразы — жёлтым #F5C518
- Большие отступы сверху и снизу (белое пространство = дизайн)
- Никаких иконок, рамок, декоративных элементов

## ПРОМТ-ШАБЛОН ДЛЯ NANO BANANA PRO

При генерации любого визуала для каруселей Влада — добавляй этот блок в конец промта:

```
STYLE REQUIREMENTS:
Format: vertical 9:16, 1080x1920px
Color grade: deep rich blacks, warm amber highlights,
             cool blue undertones in shadows
Photorealistic, editorial quality, slight film grain
Cinematic depth of field
Mood: [определяется темой слайда]
NO text overlays, NO watermarks
```

## СЦЕНАРИИ ПРИМЕНЕНИЯ

### Когда нужен визуал-обложка
Используй реальную фотографию Влада или человека из темы карусели.
Добавляй тёмный оверлей в редакторе (Canva/Figma/Unfold).
Если фото нет — генерируй через Nano Banana Pro с промтом:

```
[Описание человека/сцены].
Cinematic portrait/scene, vertical 9:16 format.
Dark moody atmosphere, deep shadows.
Color grade: warm amber highlights, cool blue shadows.
Photorealistic, editorial photography style,
35mm film grain, shallow depth of field.
No text overlays.
```

### Когда нужен визуал для смыслового слайда
Используй тёмный/чёрный фон сцены.
Объект должен быть хорошо читаем на тёмном фоне.
Промт всегда заканчивай блоком STYLE REQUIREMENTS выше.

### Текстовые слайды
Никакой генерации не нужно. Просто: чёрный фон #000000, белый текст, жёлтый акцент #F5C518.

## ЧЕКЛИСТ ПЕРЕД ПУБЛИКАЦИЕЙ

- [ ] Обложка останавливает скролл за 1 секунду?
- [ ] На каждом слайде ≤1 мысль?
- [ ] Жёлтый акцент есть, но не превышает 20% текста?
- [ ] Текстовые слайды — чистый чёрный фон?
- [ ] Нет перегруженных слайдов (>5 строк)?
- [ ] Финальный слайд: конкретный CTA, не «ставь лайк»?

## ПРИМЕРЫ УДАЧНЫХ ОБЛОЖЕК

Для этого аккаунта (тематика — отношения, интим, пары):
- Уверенная женщина, лёгкая улыбка или игривый взгляд
- Пара в нежном/динамичном взаимодействии
- Силуэт или деталь с интригой

Лучший кандидат из архива: `photo_2026-03-20 20.30.00.jpeg`
(женщина на лестнице, белый/бежевый фон → отлично берёт тёмный оверлей)
