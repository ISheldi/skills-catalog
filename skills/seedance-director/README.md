# seedance-director

Orchestrator and router for the 15 Seedance 2.0 (Higgsfield) video-prompt skills. Use as the single front door whenever the user wants a video but it is unclear which style or format fits, wants a multi-video campaign or series, asks "what kind of video should I make", or just says "make a video" generically. Triggers on - видео, сделай видео, нужен ролик, какое видео сделать, видеореклама, рекламный ролик, промт для видео, видео для продукта/бренда/бизнеса, не знаю какой формат, видео кампания, серия роликов, raskadrovka, Seedance, Higgsfield video, make a video, generate a clip, video prompt, which video skill, animate this. Routes to the right seedance-* skill - cinematic, ecommerce-ad, product-360, social-hook, motion-design-ad, brand-story, real-estate, fashion-lookbook, food-beverage, music-video, fight-scenes, anime-action, cartoon, 3d-cgi, comic-to-video.
